# Brief T4.7 - Exposicion Spot / DEXC

## Depende de

- T3.4 `exposicion_spot_mensual`

## Objetivo

Pantalla de exposicion spot y demanda excedente con simulador de derating.

## UX

- KPIs: spot legitimo, excedente pico/valle/resto, cargo DEXC.
- Simulador: reducir pico 5/10/15% y estimar cargo evitable.
- Serie mensual con cambio porcentual.

## Checks

- Simulacion no altera datos reales.
- Cargo evitable documenta formula usada.
