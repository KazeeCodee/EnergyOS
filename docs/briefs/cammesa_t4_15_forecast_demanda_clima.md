# Brief T4.15 - Forecast Demanda + Clima

## Estado

Proyecto aparte. Requiere ingesta horaria del cliente y datos climaticos.

## Objetivo

Forecast de demanda con variables climaticas.

## No empezar hasta tener

- Datos horarios del cliente.
- Historico meteorologico por sitio.
- Definicion de horizonte: dia siguiente, semana, mes.
- Metrica de error: MAPE/MAE.

## Entregable esperado

- Pipeline separado.
- Modelo versionado.
- UI de forecast con bandas de confianza.
